Abstract

Linear time-invariant systems and their system response due to various input signals. System stability.

A linear time-invariant (LTI) system is a system where
  1. the relationship between the input \(x(t)\) and output \(y(t)\) is linear
  2. if the input is shifted in time, the output is also shifted in time

The fundamental property of LTI systems is that any LTI system can be completely characterized by its impulse response \(h(t)\). The output of the system, \(y(t)\), for any input is then the convolution of the input signal \(x(t)\) and the impulse response \(h(t)\).

A system mode is a complex exponential of the form \(s_k(t) = e^{\omega t}\). where \(\omega\) is the frequency of the mode.

LTI system stability

A system is asymptotically stable if for all system modes \(s_k(t)\), \[\lim_{t\to\infty} s_k(t) = 0\]

The system is stable if all modes decay to zero as \(t\to\infty\). Since the modes are complex exponentials, the system is stable if all modes have negative real parts.

A system is marginally stable if system modes \(s_k(t)\) decay to zero but some remain constant as \(t\to\infty\). \[\lim_{t\to\infty} s_k(t) = c_k\]
A system is unstable if there exists at least one system mode \(s_k(t)\) that grows without bound as \(t\to\infty\).

Note that BIBO stability is implied by asymptotic stability, which is a stronger stability condition.

A system is bounded-input bounded-output (BIBO) stable if for any bounded input signal \(x(t)\), the output signal \(y(t)\) is also bounded. Note that both asymptoically stable and marginally stable systems are BIBO stable.
